webpack的概念解释

1. webpack基于node.js,所以webpack遵循CommonJS规范
2. webpack自身只能构建 js或者json格式的文件 (对于其他文件需要借助对应的loader或者plugin)
   2.1 备注 ： 在es6的 模块化当中，如果引入的是一个json文件，这个json文件当默认导出使用 3. webpack自身只能构建 只能解析export import语法，其他es6语法不能解析

测试

  1. webpack自身的命令 ·webpack entryFilePath -o outputFilePath --mode=development[/production]·
  2. 只能解析export 和 import语法
  3. ES6模块化语法把json文件做默认导出使用
  4. 不能解析其他格式文件
